# 模式概览

以下模式并非互斥 —— 它们可以组合使用。一个生产级编码智能体可能同时运用 `监管者 + 并行扇出 + 黑板 + 生成器-批评者 + 人在回路 + MCP`。

| # | 模式 | 类别 | 一句话概括 |
|---:|---|---|---|
| 1 | [监管者 / 管理者](./supervisor-manager) | 控制 | 主智能体规划、路由和综合；专家执行子任务。 |
| 2 | [智能体即工具](./agents-as-tools) | 控制 | 专家作为可调用工具暴露；宿主保持对话控制权。 |
| 3 | [交接 / 路由器](./handoff-router) | 控制 | 当前智能体将控制权转移给另一个，后者接管对话。 |
| 4 | [顺序管道](./sequential-pipeline) | 信息 | 任务按固定顺序通过智能体；每步的输出是下一步的输入。 |
| 5 | [并行扇出 / 收集](./parallel-fanout-gather) | 信息 | 将任务拆分到多个智能体并行执行；聚合器合并结果。 |
| 6 | [层级分解](./hierarchical-decomposition) | 控制 | 多级管理者-工作者层级；上层分解，下层执行，必要时递归。 |
| 7 | [图 / 状态机 / 工作流](./graph-workflow) | 控制 | 用显式图或状态机定义流程，而非让 LLM 即兴发挥。 |
| 8 | [群聊 / 会议](./group-chat) | 信息 | 多个智能体共享一个线程；主持人或选择器挑选下一位发言者。 |
| 9 | [嵌套对话 / 内部团队](./nested-chat) | 信息 | 智能体在向外回复前运行内部多智能体子对话。 |
| 10 | [辩论 / 裁判](./debate-judge) | 决策 | 多个智能体持对立立场，然后裁判或投票选出结果。 |
| 11 | [生成器-批评者](./generator-critic) | 决策 | 一个智能体产出；另一个批评、验证、评分或提出编辑。 |
| 12 | [精炼循环](./refinement-loop) | 决策 | 生成 → 评估 → 修订，直到满足退出条件或预算耗尽。 |
| 13 | [角色扮演 / SOP](./role-playing-sop) | 环境 | 智能体扮演 PM、架构师、开发、QA 角色，受文档化 SOP 约束。 |
| 14 | [黑板 / 共享内存](./blackboard-shared-memory) | 信息 | 智能体通过共享状态、知识库、任务板或工作空间间接协作。 |
| 15 | [事件总线 / 发布-订阅](./event-bus-pubsub) | 信息 | 智能体通过事件、主题或队列异步通信 —— 而非直接调用。 |
| 16 | [市场 / 拍卖 / 合同网](./market-auction-contract-net) | 决策 | 通过竞价、定价或合同网协议分配任务和资源。 |
| 17 | [点对点 / 群体](./peer-swarm) | 控制 | 无固定中心；智能体通过直接消息、共享环境或动态交接自组织。 |
| 18 | [智能体混合](./mixture-of-agents) | 决策 | 分层集成 —— 每层读取前一层的多个输出并改进。 |
| 19 | [人在回路](./human-in-the-loop) | 环境 | 人类是特殊智能体，用于批准、纠正、路由、中断或最终决策。 |
| 20 | [协议中介网络](./protocol-mediated) | 协议 | 通过 MCP、A2A、ACP、智能体客户端协议连接工具、智能体、客户端和平台。 |
| 21 | [边界澄清](./clarification-at-edge) | 决策 | 在智能体-智能体交接边界或不确定操作前插入澄清步骤。 |
| 22 | [协调器 / 调度器](./coordinator-dispatcher) | 控制 | 调度器将请求分发给智能体、工作流或工具，并管理任务状态、重试、超时、路由。 |
| 23 | [投票 / 集成](./voting-ensemble) | 决策 | 多个智能体的独立候选；投票、评分或验证器选出最终答案。 |
| 24 | [复合模式](./composite-pattern) | 组合 | 真实系统组合管道 + 并行 + 交接 + 批评者 + 人在回路 + 黑板 + 协议层。 |
| 25 | [工作空间 / 沙盒隔离](./workspace-isolation) | 环境 | 每个智能体在自己的工作空间、git worktree、容器或沙盒中运行，避免并发损坏。 |
| 26 | [环境媒介协作](./stigmergy-environment-mediated) | 环境 | 智能体在环境中留下痕迹（issue、todo、diff、测试结果）；其他智能体对其做出反应。 |
| 27 | [联盟 / 联邦 / 整体](./coalition-federation-holonic) | 组织 | 智能体形成临时联盟、联邦或整体 —— 治理、成员资格、自治边界。 |
| 28 | [社会模拟](./social-simulation) | 模拟 | 用长期记忆、关系、涌现行为模拟人群、组织或社会。 |
| 29 | [MARL / CTDE](./marl-ctde) | 学习 | 多智能体强化学习；集中训练，去中心化执行。 |
